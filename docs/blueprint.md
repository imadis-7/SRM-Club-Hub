# **App Name**: Assembla

## Core Features:

- Member Messaging Hub: A centralized, real-time messaging interface for club members to stay connected and coordinate effortlessly.
- Club AI Concierge: A generative AI tool designed to answer member questions about upcoming events and facilitate automated club introductions.
- Upcoming Meetings Dashboard: A focused view for members to browse upcoming scheduled meetings, join links, and specific agenda items.
- Smart Recap Archive: A library for browsing previous meeting details, enabling members to quickly catch up on missed sessions.
- AI Auto-Recap tool: A tool powered by generative AI that synthesizes recent meeting discussions and logs into clear, actionable summaries.

## Style Guidelines:

- Primary color: Electric Indigo (#4040BF), selected for its energetic and modern community-building vibe.
- Background color: Whisper Lavender (#F1F1F9), providing a crisp, sophisticated light-mode canvas.
- Accent color: Deep Ocean Blue (#2863BD), used to distinguish primary navigation elements and active meeting indicators.
- A font pairing of 'Space Grotesk' for headlines to maintain a technical, futuristic edge and 'Inter' for body text to ensure maximum legibility for long meeting summaries.
- Ultra-thin, minimalist line icons that emphasize modern aesthetics and clarity in the dashboard interface.
- A high-density sidebar-focused layout that organizes navigation between the chat, calendar, and recap galleries.
- Fluid, spring-loaded transitions when toggling between the real-time chat interface and static recap cards.
